# 🚀 AI & Data Structures Projects

A collection of mini projects built using **Python, HTML, CSS, and JavaScript** to practice Data Structures, Artificial Intelligence, Algorithms, and Web Development.

## 📌 Projects

### 📍 Graph Traversal (Python)
- BFS & DFS implementation using an adjacency list
- Object-Oriented Programming approach

**Tech:** Python

---

### 💬 Smart Customer Support Chatbot
- Rule-based chatbot
- Modern responsive UI
- Handles order tracking, refunds, delivery, and FAQs

**Tech:** HTML, CSS, JavaScript

---

### ♛ N-Queens Visualizer
- Interactive visualization of the N-Queens problem
- Backtracking with Branch & Bound
- Pause, Resume, Reset, and Multiple Solutions

**Tech:** HTML, CSS, JavaScript

---

### 📈 AI Stock Trading Expert System
- Rule-based Buy, Sell & Hold recommendations
- Uses market trend, RSI, trading volume, and news sentiment

**Tech:** HTML, CSS, JavaScript

---

### 🧩 A* Maze Solver
- Visual implementation of the A* pathfinding algorithm
- Finds the shortest path using Manhattan Distance

**Tech:** HTML, CSS, JavaScript

---
